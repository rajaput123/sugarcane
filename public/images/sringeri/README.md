# Sringeri Sharada Peetham Images

This directory contains images related to Sringeri Sharada Peetham for use in the landing page and application.

## Required Images

Please add the following images to this directory:

1. **temple-hero.jpg** - Main temple complex image (recommended: 1920x1080 or larger)
   - Used as hero background on landing page
   - Should showcase the main temple complex

2. **jagadguru.jpg** - Jagadguru image (recommended: 800x1000 or similar portrait)
   - Used for Jagadguru-related features

3. **temple-architecture.jpg** - Temple architecture detail (recommended: 1200x800)
   - Showcasing temple architecture and design

4. **festival.jpg** - Festival celebration image (recommended: 1600x900)
   - Navaratri or other festival celebrations

5. **temple-entrance.jpg** - Temple entrance image (recommended: 1200x800)
   - Main entrance or Raja Gopuram

## Image Guidelines

- Format: JPG or PNG
- Optimize images for web (use tools like ImageOptim or similar)
- Maintain aspect ratios for best display
- Ensure images are high quality but optimized for fast loading

## Fallback

If images are not available, the landing page will use gradient backgrounds as fallback.
